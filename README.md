 项目简介
本项目为多分类机器学习实验，基于Dry Bean干豆数据集，完成数据清洗、数据集划分、多分类模型训练与精度对比实验。通过多种机器学习算法对豆子品种进行自动识别，对比不同模型的分类准确率，完成模型性能评估。

 一、文件说明
1. main.py：主程序，包含数据处理、模型训练、结果评估与绘图代码
2. requirements.txt：项目运行所需要的Python第三方库
3. full_dataset.csv：完整原始数据集
4. train.csv：训练集数据
5. test.csv：测试集数据
6. 各类png图片：混淆矩阵、精度对比图、相关性热力图等实验结果图

 二、实验内容
1. 数据预处理：缺失值检查、异常值剔除，将完整数据集拆分为训练集与测试集。
2. 模型搭建：搭建逻辑回归、随机森林、XGBoost等多种分类模型。
3. 模型评估：计算各模型准确率，输出对比结果并保存实验图表。

 三、运行方式
1. 安装环境：执行 `pip install -r requirements.txt`
2. 运行代码：直接启动 main.py
3. 生成结果：自动输出精度表格与各类可视化图片

 四、实验结论
随机森林在本数据集上分类效果最优，泛化能力更强，更加适合该豆类品种识别任务。
